# 🏎️ NexDrive - Premium Automotive Marketplace

**NexDrive** is a high-end automotive landing page built with modern **JavaScript**, focusing heavily on **performance**, responsiveness, and a premium user experience. 
The project features a "High-Tech" aesthetic with a seamless dynamic theme engine.

---

## 🚀 Key Technical Features

* 🌓 **Dynamic Theme Engine:** Full Light and Dark mode switching, with user preference persistence using `localStorage`.
* ⚡ **Performance Optimized:** Achieving top-tier scores in Google Lighthouse through:
    * **WebP Image Format:** High-quality imagery with significantly reduced file sizes.
    * **Lazy Loading:** Images are loaded only as they enter the viewport to save bandwidth.
    * **Asset Minification:** Compressed CSS and JS files for faster parsing and execution.
* 📊 **Data-Driven UI:** Vehicle listings, special features, and client reviews are dynamically fetched from external **JSON** files via the `Fetch API`.
* 📱 **Fully Responsive:** A tailored browsing experience across all devices (Mobile, Tablet, Desktop) powered by **Swiper.js**.
* 🖱️ **Interactive Parallax:** Sophisticated visual effects in the Hero section that provide a modern, interactive feel on desktop devices.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (using Custom Variables/Root).
* **Logic:** JavaScript (ES6+), Async/Await, DOM Manipulation.
* **Libraries:** * [Swiper.js](https://swiperjs.com/) - For responsive, touch-enabled car sliders.
    * [FontAwesome 6](https://fontawesome.com/) - For scalable vector icons.
* **Architecture:** JSON-based local API structure for scalable content management.
